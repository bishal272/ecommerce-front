## Getting Started

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/bishal272/ecommerce-front.git
   ```
2. **Install Dependencies:**

   ```bash
   cd your-eCommerce-app
   npm install
   ```

3. **Run the App:**

   ```bash
   npm run dev
   ```

4. **Open Your Browser:**
   Open http://localhost:3000 to view the web app.
